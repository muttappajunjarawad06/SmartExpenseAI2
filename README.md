# SmartExpenseAI Agent

## Problem
Managing personal expenses is difficult. SmartExpenseAI helps users analyze spending and get budget suggestions.

## Features
- Expense tracking
- Spending category analysis
- Budget recommendations
- AI Agent workflow
- MCP tool integration

## Technology
- Python
- Google ADK
- Gemini AI
- MCP Server
- Pandas

## How to Run

Create environment:

python -m venv venv

Activate:

venv\Scripts\activate

Run:

python app.py

## Example

Input:
Analyze my expenses

Output:
Expense tracking completed
Spending categories analyzed
Budget suggestions generated
